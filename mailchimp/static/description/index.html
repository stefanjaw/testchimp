Features Highlights

Import Data

Import List/Audiences, Members, Templates, Merge Fields, Segments and Campaigns from MailChimp to Odoo.

Export Data

Export List/Audiences and Members to MailChimp.

Manage Campaigns

Allows to test before sending Campaign and Schedule it or directly send it by selecting templates and filtering receipts with saved Segments.

Statistics of Campaigns

Get the statistics of sent Campaigns back to Odoo. So, that you can trace sent, open, clicked and replied count for each Campaign.

Support Webhook

Get member update real-time by configuring Webhooks!

Connect Multiple Accounts

Integrate and Manage multiple MailChimp account with single Odoo instance.
